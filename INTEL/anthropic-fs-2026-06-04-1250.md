# Anthropic/financial-services — 仓库分析

## BLUF
- Anthropic 官方 FSI（金融服务）垂直落地仓库：把"投行/行研/PE/财富管理"工作流打包成 **可即装的 Claude 插件 + 可托管部署的 Managed Agent** 两种形态。
- 核心策略：用 **MCP 整合 11 家头部金融数据源**（Daloopa、Morningstar、FactSet、Moody's、LSEG、PitchBook、S&P Global、Chronograph、MT Newswires、Aiera、Box）作为壁垒，把 Claude 锁进金融分析师的桌面。
- 不是 demo：是 **生产级参考模板**（Apache-2.0），明确合规免责声明（不出投资建议、不下单、需人工复核），瞄准企业 IT + 合规采购流程。

## 仓库描述（约 200 字）
anthropics/financial-services 是 Anthropic 于 2026 年 2 月从 fsi-plugins-dev 移植并公开的金融服务垂直 Claude 工具集，仓库按"agent / skill / command / connector"四层文件化组织，全部为 markdown + JSON（无构建步骤），使用 Apache-2.0 协议。核心包含 8 个垂直插件（financial-analysis、investment-banking、equity-research、private-equity、wealth-management、fund-admin、operations 及 LSEG/S&P 合作方插件），覆盖 Pitch Agent、Meeting Prep Agent、GL Reconciler 等端到端工作流。所有 connector 通过 MCP 协议统一接入，仓库还附带 `claude-for-msft-365-install` 工具，方便企业 IT 在自有云（Vertex/Bedrock/内网网关）部署 Claude for Microsoft 365 加载项。已获 ~29.8k stars、4.2k forks，60 次主分支提交，月度活跃迭代。

## README 核心要点
1. **双形态交付**："Cowork 插件" + "Managed Agents API（`/v1/agents`）" 同源同 prompt，机构选择本地运行或托管运行。
2. **Agents（命名工作流）**：
   - Pitch Agent — Comps/Precedents/LBO → 品牌化 pitch deck
   - Meeting Prep Agent — 客户会前简报
   - Market Researcher、GL Reconciler 等端到端角色
   - 每个 agent 是 self-contained 插件，自带 skills
3. **Vertical Plugins**（按业务条线拆分）：
   - financial-analysis（核心）：Comps、DCF、LBO、3-statement、deck QC、Excel audit + **全部 11 个数据 connector**
   - investment-banking：CIMs/teasers/process letters/merger models
   - equity-research：earnings notes/initiations
   - private-equity：sourcing → IC memos → portfolio monitoring
   - wealth-management：client reviews/rebalancing/TLH
   - fund-admin：GL recon/break tracing/NAV tie-out
   - operations：KYC 文档解析
   - lseg、sp-global：合作方垂直
4. **MCP Integrations**（11 家）：Daloopa、Morningstar、S&P Global (kfinance.kensho.com)、FactSet、Moody's、MT Newswires、Aiera、LSEG、PitchBook、Chronograph、Egnyte、Box — 全部集中在 financial-analysis core。
5. **架构层级**：
   - Agents = `plugins/agent-plugins/<slug>/`
   - Skills = vertical 源 + agent bundled 同步
   - Commands = `/comps` `/dcf` `/earnings` 等显式斜杠
   - Connectors = `plugins/vertical-plugins/financial-analysis/.mcp.json`
   - Managed-agent wrappers = `managed-agent-cookbooks/<slug>/`
6. **Claude for Microsoft 365**：独立加载项部署工具，针对 Vertex AI / Bedrock / 内网 LLM 网关，IT admin 走 `claude plugin install` + `/setup` 流程。
7. **强免责声明**：明确"非投顾/法务/税务/会计建议"，所有产物需"持牌专业人士"复核；不执行交易、不挂账、不审批 onboarding。
8. **可定制**：换 connector、添加 firm terminology、注入品牌 PPT 模板、复制结构扩展新工作流。
9. **贡献机制**：`python3 scripts/check.py` 做 manifest lint + 跨文件引用校验，CI 强制 sync-agent-skills.py 防 skill 漂移。

## 隐含战略意图（约 150 字）
Anthropic 正以"行业垂直开箱即用"为差异化武器，正面阻击 OpenAI 在企业级落地的攻势。**真正护城河是 MCP 整合 11 家金融数据 provider** — 这相当于把 Claude 变成"金融版 Excel"的中枢，绑定 Daloopa/PitchBook/FactSet/Moody's 等企业已有的订阅开支。**双形态交付（Cowork + Managed Agents）** 降低 IT 部署摩擦，**MSFT 365 加载项**直接进入投行/行研 Excel 工作流。**强合规免责声明 + Apache-2.0 + 强调 human-in-the-loop** 显示打法是先渗透到"分析师桌面"再向上攻 IBOR/OMS 决策层 — 这是用 LLM 公司罕见的"慢而稳"打法。
