## BLUF

- ESP32-S31 = 乐鑫双核 RISC-V 旗舰 SoC,主打"Wi-Fi 6 + 多协议 + HMI"一体化,定位中高端 IoT/智能显示。
- 与 S3 的最大差异是新增 **Wi-Fi 6 (802.11ax)**、**1 Gbps Ethernet MAC**、**双核异构 (128-bit SIMD)** 与 **PUF 安全**;较 P4 在无线协议栈与 PSRAM 扩展上更完整。

## ESP32-S31 规格 (150字)

双核 32-bit RISC-V @320 MHz,6.86 CoreMark/MHz,其中一个核带 128-bit 数据通路与 SIMD。无线:2.4 GHz Wi-Fi 6 (802.11ax)、IEEE 802.15.4 (Thread/Zigbee)、Bluetooth 5.4 LE (含 LE Audio / Direction Finding / Mesh 1.1) + Classic BR/EDR。有线:1000 Mbps Ethernet MAC。存储:512 KB SRAM、250 MHz 8-bit DDR PSRAM 接口 (Octal SPI)。HMI:DVP 8-16 bit 摄像头、LCD 8-24 bit RGB/I8080/MOTO6800、JPEG/PPA/2D-DMA 加速、14 通道触摸。音频:双 I2S + 硬件级 BT 音频同步。安全:TRNG、RAM PUF、Secure Boot、Flash/PSRAM 加密、AES-128/256、RSA/ECDSA/ECC、TEE + APM。60 GPIO。开发框架:ESP-IDF、ESP-Matter、ESP-BLE-AUDIO、ESP-GMF,并对接 ESP Private Agents (LLM Agent 平台)。

## 与 S3 / P4 差异 (50字)

相对 S3:S31 升级到 Wi-Fi 6、加 1G Ethernet、双核异构+SIMD、PUF 安全。相对 P4:S31 协议栈更全 (Wi-Fi 6 + BT 5.4 LE + 15.4)、HMI 加速器更丰富、保留 Octal PSRAM 扩展,但 P4 偏多核高性能,S31 偏无线 + 多媒体一体。
